Welcome
===================================================

This is the documentation for `RoboJackets' <https://robojackets.org/>`_ RoboCup
Software team. We are a primarily undergraduate robotics team from Georgia Tech
who compete in Division B of the RoboCup Small Size League. See the `RoboCup SSL
website <https://ssl.robocup.org/about/>`_ for more details on the competition.

.. note::

   This project is under active development.
