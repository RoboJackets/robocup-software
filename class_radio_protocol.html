std::function<std::vector<uint8_t>const rtp::ControlMessage* msg)> RadioProtocol::rxCallback

Callback that is called whenever a packet is received.

Set this in order to handle parsing the packet and creating a response. This callback should return a formatted reply buffer, which will be sent in the appropriate reply slot.
